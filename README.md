# 🎨 Portfolio Assets

Repositório destinado ao armazenamento de imagens, ícones e recursos gráficos utilizados na documentação dos projetos, no perfil do GitHub e em materiais relacionados ao portfólio.

## 📂 Conteúdo

Este repositório centraliza arquivos utilizados em:

* README dos projetos
* README do perfil do GitHub
* Documentações técnicas
* Diagramas e materiais de apoio

## 🖼️ Recursos Disponíveis

Atualmente o repositório contém ícones e imagens relacionados a:

* Zabbix
* Proxmox VE
* VMware
* VirtualBox
* Fedora Linux
* Zorin OS
* The Dude

## 📌 Objetivo

Manter os recursos gráficos organizados em um único local, facilitando a reutilização entre diferentes projetos e garantindo consistência visual na documentação.

## ℹ️ Observação

Este repositório possui finalidade exclusivamente documental e não contém código-fonte ou aplicações executáveis.
